第一天详细导航

第一天路线地图

A: 故宫 → B: 景山公园 → C: 王府井

上午： 天安门东地铁站 → 天安门广场 → 故宫午门入口

沿途景点： 人民英雄纪念碑、毛主席纪念堂

故宫游览路线： 午门 → 太和殿 → 中和殿 → 保和殿 → 乾清宫 → 交泰殿 → 御花园 → 神武门

下午： 神武门 → 景山公园 → 景山万春亭(俯瞰故宫全景) → 步行至王府井大街

晚上： 王府井小吃街 → 王府井步行街购物 → 东华门地铁站

交通信息

公共交通：

  • 地铁1号线至天安门东站下车，步行约10分钟至故宫午门
  • 故宫神武门至景山公园步行约5分钟
  • 景山公园至王府井步行约20分钟或乘坐公交103路至王府井站

自驾路线：

  • 请注意，天安门广场周边限行且停车位紧张，建议将车停在附近停车场后步行或乘坐公共交通前往
  • 推荐停车点：北京市地坛体育馆地下停车场，停车后可乘坐地铁5号线至天安门东站

步行距离与时间：

  • 地铁站至故宫：约800米，步行约10分钟
  • 故宫内全程游览：约3-4公里，需时约3-4小时
  • 故宫至景山公园：约400米，步行约5分钟
  • 景山公园至王府井：约1.5公里，步行约20分钟

第二天详细导航

第二天路线地图

A: 八达岭长城 → B: 奥林匹克公园(鸟巢)

上午： 北京北站 → 乘坐S2线 → 八达岭长城

长城游览建议： 北八楼方向(人少风景好) → 九眼楼(最高点) → 老龙头 → 八达岭长城博物馆

午餐： 八达岭服务区用餐

下午： 乘坐S2线返回 → 转乘地铁8号线 → 奥林匹克公园

奥林匹克公园游览： 鸟巢(国家体育场) → 水立方(国家游泳中心) → 奥林匹克塔 → 奥林匹克森林公园

晚上： 奥林匹克公园站 → 返回住宿地

交通信息

公共交通：

  • 北京北站乘坐S2线火车至八达岭站，约1.5小时
  • 或乘坐877路、919路长城专线直达八达岭长城，约1.5-2小时
  • 返程后，地铁8号线至奥林匹克公园站下车，步行即可到达鸟巢

自驾路线：

  • 从市区出发→京藏高速→延庆方向→八达岭长城出口下
  • 全程约70公里，行车时间约1.5小时
  • 长城至奥林匹克公园约65公里，行车时间约1小时20分钟
  • 八达岭长城停车场：停车费约15元/小时
  • 奥林匹克公园停车场：停车费约10元/小时

距离与时间：

  • 市区至八达岭长城：约70公里
  • 长城游览：约3公里，需时约2-3小时
  • 八达岭长城至奥林匹克公园：约65公里
  • 奥林匹克公园游览：约2公里，需时约2小时

第三天详细导航

第三天路线地图

A: 颐和园 → B: 天坛 → C: 什刹海/南锣鼓巷

上午： 地铁4号线 → 北宫门站 → 颐和园

颐和园游览路线： 东宫门 → 仁寿殿 → 长廊 → 石舫 → 佛香阁 → 苏州街 → 西堤

午餐： 颐和园附近餐厅

下午： 地铁4号线 → 转乘5号线 → 天坛东门站 → 天坛公园

天坛游览路线： 东门 → 圜丘坛 → 皇穹宇 → 祈年殿 → 回音壁 → 三音石 → 北门

晚上： 地铁5号线 → 什刹海/南锣鼓巷 → 胡同文化体验 → 返回住宿地

交通信息

公共交通：

  • 地铁4号线至北宫门站下车，步行约500米至颐和园
  • 颐和园游览结束后，乘坐地铁4号线至宣武门站，换乘5号线至天坛东门站
  • 天坛游览结束后，乘坐地铁5号线至北新桥站或雍和宫站，步行至什刹海或南锣鼓巷

自驾路线：

  • 市区→中关村→颐和园路→颐和园东门停车场
  • 颐和园→二环路→永定门内大街→天坛公园
  • 全程约25公里，行车时间约1小时
  • 颐和园停车场：停车费约15元/小时
  • 天坛公园停车场：停车费约10元/小时
  • 什刹海附近建议停车在鼓楼大街停车场，后步行游览

距离与时间：

  • 市区至颐和园：约15公里，驾车约40分钟
  • 颐和园游览：约3-4公里，需时约3小时
  • 颐和园至天坛：约18公里，驾车约50分钟
  • 天坛游览：约2公里，需时约2小时
  • 天坛至什刹海：约7公里，驾车约25分钟

第一天：故宫

A. 故宫博物院

地址: 北京市东城区景山前街4号

电话: 010-85007421

评分: 4.8

推荐游览时间: 4-5小时

交通: 地铁1号线天安门东站下车步行

门票: 淡季（11月1日-次年3月31日）40元/人，旺季（4月1日-10月31日）60元/人

推荐理由: 世界上规模最大的宫殿型建筑群，中国明清两代的皇家宫殿，世界文化遗产

游览建议

早上8:30到故宫，从午门入口进入，沿中轴线参观太和殿、中和殿、保和殿等主要宫殿
中午在故宫内的咖啡厅或附近餐厅用餐
下午参观东路的乾清宫、交泰殿、御花园等
参观完故宫后可以登上景山公园，俯瞰紫禁城全景
晚上可以前往王府井或前门大街，品尝北京特色小吃，体验老北京风情

第二天：长城和鸟巢

B. 八达岭长城

地址: 北京市延庆区G6京藏高速58号出口

电话: 010-69121383

评分: 4.6

推荐游览时间: 3-4小时

交通: 乘坐S2线火车或长城专线巴士

门票: 淡季（11月1日-次年3月31日）35元/人，旺季（4月1日-10月31日）40元/人

推荐理由: 明长城中保存最好的一段，也是最早对外开放的长城景区

E. 鸟巢（国家体育场）

地址: 北京市朝阳区国家体育场南路1号

电话: 010-84373008

评分: 4.5

推荐游览时间: 1-2小时

交通: 地铁8号线奥林匹克公园站下车

门票: 50元/人

推荐理由: 2008年北京奥运会主会场，独特的建筑风格使其成为北京的新地标

游览建议

早上7:00出发前往八达岭长城，避开高峰期
建议从北八楼方向攀登，人相对较少，风景也更好
午餐可以在长城附近的餐厅用餐
下午返回市区，前往奥林匹克公园参观鸟巢和水立方
晚上可以在奥林匹克公园周边用餐，欣赏鸟巢和水立方的夜景

第三天：颐和园和天坛

C. 颐和园

地址: 北京市海淀区新建宫门路19号

电话: 010-62881144

评分: 4.7

推荐游览时间: 3-4小时

交通: 地铁4号线北宫门站或西苑站下车

门票: 淡季（11月1日-次年3月31日）20元/人，旺季（4月1日-10月31日）30元/人

推荐理由: 中国现存规模最大、保存最完整的皇家园林，世界文化遗产

D. 天坛公园

地址: 北京市东城区天坛东路1号

电话: 010-67028866

评分: 4.6

推荐游览时间: 2-3小时

交通: 地铁5号线天坛东门站下车

门票: 淡季（11月1日-次年3月31日）10元/人，旺季（4月1日-10月31日）15元/人

推荐理由: 世界上规模最大、环境最美的古代祭天建筑群，世界文化遗产

游览建议

早上8:00前往颐和园，从东宫门进入，游览长廊、佛香阁、昆明湖等主要景点
中午可以在颐和园附近用餐
下午前往天坛公园，参观祈年殿、回音壁、三音石等著名景点
晚上可以前往什刹海或南锣鼓巷，体验老北京胡同文化
如有时间，可以欣赏一场京剧或杂技表演，作为北京之行的完美收官

旅行小贴士

最佳旅游季节： 北京的春季（4-5月）和秋季（9-10月）气候宜人，是旅游的最佳时节。

交通卡： 建议购买北京交通一卡通，可用于地铁、公交等公共交通。

景点门票： 热门景点建议提前在网上预订门票，特别是故宫需要提前预约。

当地美食： 不要错过北京烤鸭、炸酱面、涮羊肉、豆汁、艾窝窝等北京特色美食。

购物： 王府井、秀水街、潘家园是著名的购物区，可以购买丝绸、景泰蓝、剪纸等纪念品。

紧急电话： 报警：110，救护：120，旅游投诉：12301。
